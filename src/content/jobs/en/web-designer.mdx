Web designers play an important role in software product design, they are responsible for designing and implementing user interface, as well as user experience. They need to collaborate with product managers, developers, testers, etc. to ensure the success of the product.
Web designers need to have the following skills:

- Have a deep understanding of user experience and interaction design, including user research, user testing, user flow, user interface design, etc.
- Possess certain aesthetic ability and artistic cultivation, innovative thinking and research ability
- Able to assist product managers to complete product prototypes and interactions, gain insight into user needs
- Excellent communication and presentation skills, able to work with developers, testers, etc.
- Familiar with Photoshop, Illustrator, Figma, Sketch and other design tools
- Familiar with HTML, CSS, JavaScript and other web technologies
- Have a certain understanding of mobile terminal design, including iOS, Android, WeChat applets, etc.
