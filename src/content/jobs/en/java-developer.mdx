Java engineers are mainly responsible for Java-related product development. Considering the wide application of Java and related ecology, Java engineers need to have the following abilities:

* Familiar with Java/Kotlin language, able to use Java/Kotlin language for development
* Understand mainstream Java frameworks, such as Spring, Spring Boot, Spring Cloud, Quarkus, etc.
* Understand mainstream Java development tools, including Maven, Gradle, IntelliJ IDEA, Docker, etc.
* Understand databases and related technologies, such as MySQL, PostgreSQL, MongoDB, Redis, etc.
* Understand related technologies of distributed systems, such as distributed communication, service registration discovery, load balancing, etc.
* Have a certain understanding of microservice architecture, understand domain-driven design, event-driven, cloud native and other concepts
* Have a certain understanding of the Java ecosystem, such as Jakarta EE, Kafka, etc.
* Have a certain understanding of Java third-party development kits, such as Apache Commons, Guava, Lombok, etc.
* Have a certain understanding of Kotlin third-party development kits, such as Kotlin Coroutines, Kotlin Flow, Kotlinx Serialization, etc.
* Understand Java performance tuning, such as JVM tuning, GC tuning, etc.
* Understand the cutting-edge trends related to Java, such as GraalVM Native Image, Virtual Threads, etc.
