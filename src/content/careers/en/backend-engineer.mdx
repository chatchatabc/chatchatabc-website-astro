A Back End Engineer is responsible for curating the structure of server-side information. Back Side Engineers primarily write server scripts and APIs to be utilized by front-side engineers and UX designers. The day-to-day consists of optimizing servers for speed and stability, implementing security structures, generating reusable code libraries, and generating data storage solutions.

Qualified back-end engineers need to have the following abilities:

- Have a good understanding of a certain development language and be able to use the language for development proficiently.
- Have an in-depth understanding of the database, and be able to use the database for data storage skillfully.
- Have an in-depth understanding of network protocols, and use various network protocols for communication skillfully.
- Have a certain understanding of software engineering and ensure the code quality of the software.
- Have a certain understanding of software architecture and be able to design a reasonable software architecture.
- Have a certain understanding of software security and be able to integrate software security into software development.
- Have a certain understanding of software performance, and be able to prioritize software performance in daily development.
- Have a certain understanding of continuous integration and integrate continuous integration into software development.
- Have a certain understanding of the operation and maintenance of the online environment, and be able to complete the operation and maintenance of the online environment independently.
